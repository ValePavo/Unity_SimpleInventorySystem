# Simple Inventory System – Unity

This project showcases a simple inventory system developed in Unity using C#.  
It includes item pickup, dropping items, and basic inventory management mechanics.

## Overview

- Focused example project demonstrating gameplay programming in Unity.
- Implemented using clean, modular C# scripts.
- Designed to be easily extendable for future features.

## Features

- Pickup and drop items in the world
- Inventory UI displaying collected items
- Modular architecture using reusable components
- Clean separation between data and logic
- Optimized for real-time gameplay

## Controls

- **I** – Open/Close inventory
- **Right-click** – Drop selected item
